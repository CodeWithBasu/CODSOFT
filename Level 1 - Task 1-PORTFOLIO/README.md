# Level 1 - Task 1: Personal Portfolio

A premium, professional personal portfolio built as part of the **CODSOFT Web Development Internship**. This project showcases a modern "Cyber-Modern" aesthetic with advanced CSS techniques and a focus on user experience.

## 🌟 Live Preview
The project is designed to be hosted on any static web server. 
- **Local Dev Server:** `http://localhost:8080`

## 🎨 Design Features
- **Modern Dark Theme**: Using a deep slate palette with subtle glowing gradients.
- **Glassmorphism UI**: High-fidelity translucent cards with blurred backgrounds and thin borders.
- **Micro-Animations**: Smooth entry animations triggered on scroll using the Intersection Observer API.
- **Fully Responsive**: Optimized for all devices—from 4K desktops to small mobile screens.
- **SEO Optimized**: Includes proper semantic HTML5 tags and meta descriptions.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with variables, Grid, Flexbox, and complex animations.
- **JavaScript**: Minimalist script for scroll animations.
- **Font Awesome**: Professional icons.
- **Google Fonts**: "Outfit" typography for a premium feel.

## 🚀 How to Run Locally

### Method 1: Direct Open (Easiest)
Simply locate the `index.html` file in the project folder and double-click it to open it in your default web browser.

### Method 2: Command Line (Windows)
Open your terminal in the project directory and run:
```powershell
start index.html
```

### Method 3: Local Dev Server (Recommended for development)
If you have [Node.js](https://nodejs.org/) installed, you can run a local server to ensure all assets load correctly without CORS issues:
```bash
# Using npx (no installation required)
npx http-server . -p 8080
```
Then visit `http://localhost:8080` in your browser.
